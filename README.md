# Unemployment Analysis with Python

## Description
This project analyzes unemployment rate trends using Python.
It focuses on identifying patterns, seasonal trends, and the
impact of Covid-19 on unemployment rates in India.

## Dataset
- Automatically generated unemployment dataset
- Monthly data from 2019 to 2021

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
1. Data cleaning and preprocessing
2. Exploratory Data Analysis (EDA)
3. Overall unemployment trend visualization
4. Covid-19 impact analysis (Pre vs Post Covid)
5. Seasonal unemployment trend analysis
6. Region-wise unemployment distribution
7. Key insights extraction

## Key Insights
- Unemployment peaked during the Covid-19 period
- Post-Covid recovery trend observed
- Seasonal variations identified across months
